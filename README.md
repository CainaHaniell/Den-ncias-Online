
# Sistema de Denúncias Online

Bem-vindo ao **Sistema de Denúncias Online**! Esta aplicação web foi desenvolvida para permitir que qualquer pessoa registre denúncias de maneira simples e segura, com foco em situações como **bullying, vandalismo, violência** e outros tipos de ocorrências.

---

## 📚 Projeto Acadêmico

Este projeto foi criado como parte de um **trabalho acadêmico** na disciplina de Programação Web II, com o objetivo de aplicar conhecimentos práticos em **HTML, CSS e PHP** no desenvolvimento de uma solução com impacto social.

---

## 🚀 Tecnologias Utilizadas

* **HTML:** Responsável pela estrutura das páginas e formulários.
* **CSS:** Utilizado para o design visual, tornando a aplicação clara e acessível.
* **PHP:** Gerencia o processamento das denúncias, simula o armazenamento dos dados.

---

## ✨ Funcionalidades

* **Formulário de Denúncia:** Os usuários podem relatar situações como:

  * Bullying
  * Vandalismo
  * Violência
  * Outros tipos de problemas
* **Opção de Envio Anônimo:** O usuário pode decidir se quer se identificar ou não.
* **Confirmação de Envio:** Após preencher o formulário, o sistema confirma que a denúncia foi registrada com sucesso.
* **Design Responsivo (opcional):** O layout pode ser adaptado para funcionar bem em celulares, tablets e computadores
