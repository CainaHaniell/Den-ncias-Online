Orientações Importantes:

ATENÇÃO: Se você estiver em uma situação de perigo ou sofrer algum tipo de violência, ligue imediatamente para o número 190 e peça ajuda! Caso não esteja mais em risco e deseje registrar um Boletim de Ocorrência (BO), você pode acessar a Delegacia Interativa ou procurar a delegacia mais próxima.

Este espaço foi criado exclusivamente para o envio de denúncias relacionadas ao ambiente escolar. Se você presenciou ou tem informações sobre situações de violência, bullying, assédio ou outras ocorrências, este é o local adequado para registrar os fatos de forma segura e, se desejar, anônima.

Quanto mais informações forem fornecidas, mais rápido será possível compreender e resolver a situação, contribuindo para um ambiente escolar mais seguro para todos.

Procure incluir o máximo de dados possíveis, como nomes, apelidos, características físicas, redes sociais (Facebook, Instagram, TikTok etc.), veículos envolvidos (com placas ou descrição), endereços ou pontos de referência, entre outros detalhes relevantes.

Os campos marcados com * são obrigatórios.

O anonimato é garantido. No entanto, caso você queira se identificar para fornecer mais detalhes ou acompanhar a denúncia, é possível informar um telefone ou e-mail no campo de mensagem.
